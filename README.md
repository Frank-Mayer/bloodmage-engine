## Build tool

[![Yab Project](https://img.shields.io/badge/Yab_Project-2C2D72?logo=lua)](https://github.com/Frank-Mayer/yab)

- Install dependencies: `yab install`
- Run: `yab run`
- Start Editor `yab edit -- [level path]`
- Package for deployment: `yab pack`
